# KallsupToPhoniebox
Kallsup von Ikea bietet eine kostengünst Hardware f+r eine Bluetooth-Box mit Gehäuse. Diese soll um die Funktionalität von RFID-Steuerung erweitert werden

Vorteil:
 - bietet Gehäuse
 - beitet Akku
 - USB-C Ladeanschluss

Ideen:

vielleicht kann man den Lautsprecher und die Stromversorgung doppelt nutzen:
  mit einer zusätzlichen Platine für die Phonie-Funktion
